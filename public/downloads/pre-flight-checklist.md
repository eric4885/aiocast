# Podcast pre-flight checklist (toolai.my)

Use this before every recording session — solo or remote.

## Before you hit record

- [ ] **Room noise under control** — aim for ambient noise roughly below ~40 dB SPL at the mic position (quiet room; HVAC off or away from vents when possible).
- [ ] **One mic per voice** — each host/guest on their own track when using Riverside, Squadcast, or local multitrack.
- [ ] **Gain staging** — peaks around −12 to −6 dBFS on speech; leave headroom.
- [ ] **Headphones on** — closed-back for guests to avoid bleed into the mic.

## During the session

- [ ] **Clap or slate** — sync scratch track if recording separate devices.
- [ ] **Monitor levels** — watch for clipping or whisper-quiet dialogue.

## After recording

- [ ] **Backup twice** — cloud + external disk or second folder before editing.
- [ ] **First pass = listen** — one full listen before destructive cuts.
- [ ] **Filler pass after structure** — remove ums only once the story arc is locked.
